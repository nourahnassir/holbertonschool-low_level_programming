# C - malloc, free

هذا المشروع يركز على فهم الذاكرة الديناميكية (Dynamic Memory Allocation) في لغة C، وكيفية استخدام الدالتين `malloc` و `free` لإدارة الذاكرة بفعالية.

## المهام المنجزة:
* **0-create_array.c**: دالة تقوم بإنشاء مصفوفة من الحروف وتجهيزها بحرف معين.

## المتطلبات:
* اللغة: C
* الأسلوب: Betty style
* الأدوات: valgrind (للتأكد من عدم وجود تسريبات في الذاكرة)
